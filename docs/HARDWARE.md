# Hardware — mango

**Have:** Pi 5 8GB CanaKit · 128GB SD · USB gamepad · phone · TV

- **SD card:** flash on Mac via Imager, then insert in Pi underside slot
- **Gamepad:** USB receiver in Pi — primary TV remote
- **Phone:** mic + companion app (later, over WiFi)
